# JobTailor (MVP)

JobTailor is a streamlined automation tool designed to eliminate the friction of job hunting. Instead of manually editing your CV and writing cover letters for every single application, JobTailor leverages AI to instantly align your professional background with specific job descriptions.

> Status: 🚧 This project is currently an MVP in active development. Features are being added and refined daily.

---

- The Goal
The goal is simple: Zero-input tailoring. By separating your "Master Data" from the "Job Context," the application intelligently generates optimized application materials, allowing you to focus on the interview rather than the paperwork.

- Tech Stack
* Frontend: [Next.js](https://nextjs.org/) (App Router) for a fast, responsive user interface.
* Backend: [Express.js](https://expressjs.com/) for robust API handling and AI orchestration.
* Database: NoSQL for flexible, document-based storage of professional records.
* Styling: [Tailwind CSS](https://tailwindcss.com/) + UI components for a clean, professional aesthetic.
* AI Engine: Powered by advanced LLMs for context-aware content generation.

- Key Features (In Development)
* Instant Optimization: Generate tailored CVs and Cover Letters from a single text input.
* Company Intelligence: Get a quick research summary on the company before you apply.
* Master Profile Management: Maintain a single source of truth for all your professional milestones.
* Live Preview: Visualize your tailored documents in real-time before exporting.
